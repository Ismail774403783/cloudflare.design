Acronyms

Define an acronym the first time you use it on each page. Some acronyms, however, are so widely used that they do not need to be defined. See the word list for specific acronyms and definitions.

Never use a technical or Cloudflare acronym without defining it the first time you use it. Customers don’t use every product or feature, so they may not see it elsewhere.
